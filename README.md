# Vagrant
Цей проєкт містить Vagrantfile, який автоматично створює віртуальну машину на базі Ubuntu та налаштовує середовище для запуску Spring Boot застосунку.

## Що робить ця конфігурація

- Створює віртуальну машину на основі Ubuntu 22.04.
- Налаштовує приватну мережу з IP-адресою 192.168.100.100.
- Виділяє віртуальній машині 2 CPU та 4 GB оперативної памʼяті.
- Виконує автоматичне встановлення OpenJDK 17 після першого запуску.
- Синхронізує локальну директорію проєкту з директорією /vagrant у VM.
- Після завершення запуску VM автоматично запускає Spring Boot застосунок через Gradle командою bootRun.

## Призначення
Цей Vagrantfile створений для автоматичного розгортання середовища розробки, у якому Spring Boot застосунок піднімається без ручного налаштування. Це дозволяє запускати проєкт у передбачуваному та ізольованому середовищі.
